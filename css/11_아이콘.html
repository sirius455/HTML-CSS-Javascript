# CDN (Conetent Delivery Network)

  • 컨텐츠 네트워크를 통해 제공하는 것
  • 컨텐츠 제공 회사가 제공하는 여러가지 정적 자원(글꼴, 아이콘, 이미지, JS, CSS, 등)들을 가져다 사용할 수 있다
